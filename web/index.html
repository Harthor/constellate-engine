The 5 types of constellations

Constellate groups ideas into patterns of 3-6 items that reveal non-obvious connections. Each constellation falls into one of five categories:

Chain

Ideas that form a logical or causal progression nobody designed on purpose. Each step enables or extends the next.

Example: Proprietary product → open-source clones → cheaper alternatives → users migrating.

Triangulation

Three ideas that illuminate the same deep phenomenon from different angles. No single one names it; together they do.

Example: A post refusing apps + a data privacy scandal + AI agent sandboxes → the silent collapse of user agency.

Convergence

Ideas from topically distant domains that unknowingly point to the same underlying problem or structural solution.

Example: A distributed systems paper + an organizational resilience article → both describing fault tolerance under stress.

Absence

A logical gap. Given what's in a cluster, this idea should be there but isn't. Constellate detects what's missing.

Example: AI coding agents + audit tools + regulation concerns, but no standardized logging for agent decisions.

Spectrum

Four to five ideas that represent different positions on the same axis of debate or design. The insight is the axis itself.

Example: From fully autonomous AI to fully supervised AI, with intermediate positions along the control spectrum.
